Understanding Grid System

Back to Top

This template use bootstrap grid to make it very easy for anybody to understand the grid system. There is some basic details about its html structure below.

Parent Classes of Sections

.lib-blog-post class is used to creating the blog listing view.

.lib-blog-2 class is used for the blog to have another view.

.lib-blog-2 lib-blog-3 class is used for third listing view.

.lib-blog-detail class is used for the blog details where you can show the blog in detailed view.

.best-seller-pro is used to displaying the best seller books.

.lib-book-detail This class gives you show the book detail page.

.kode-banner This class is used for setting up the main banner

.kode-author This class is used for showing the books Author.

.kode-author kode-author-2 This class is for showing the books Author with another beautiful style.

.lib-event-listing This class is for showing the events of your site in listing view.

.lib-event-detail This class is for the showing the detail view of the events.

.kode-event-detail This class is for showing another view of the event detail page.

.kode-blog-list This class is for news events style in your site.

.kode-blog-list option-2 This class is for showing another view of the New events page.

.kode-portfolio-page class is used for the portfolio in your site.

.coming-soon class is used for creating the coming soon page of your site.

.error-404 class is used for the error 404 page creation.

Javascript Files

Back to Top

This template uses just 14 .js files in total. See list below to understand which files are used and why.

File Name Description
bootstrap.min.js A set of file that is used for the working of the Bookstrap grid files.
bootstrap-slider.js This file is used to running the bootstrap built in slider.
functions.js A file that contains all of the custom js code written here.
jquery.accordion.js A set of file that is used to run of the accordion tabs.
jquery.bxslider.min.js Jquery files that are used to run the bx slider in your site.
jquery.countdown.js A set of file that are used to run the js countdown counter.
jquery.counterup.min.js A set of file that is used for the working of the counter
jquery.js A set of file that is used for the working of the jquery scripts
jquery.min.js A set of file that is used for the working of the jquery min files.
jquery.plugin.js These files are used to run the jquery plugins at your site.
jquery-filterable.js A set of file that is used for the working of the filterable at your site.
map.js A set of file that is used for the working of the Maps
masonry.pkgd.min.js A set of file that is used for the working of the masnory styles.
owl.carousel.js A set of file that is used for the working of the owl slider.
